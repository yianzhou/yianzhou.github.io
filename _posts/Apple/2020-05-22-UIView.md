# 圆角

`cornerRadius` 的官方解释： By default, the corner radius does not apply to the image in the layer’s contents property; it applies only to the background color and border of the layer.

对于 `UIView`, `UIButton`, `UIScrollView`, `UITextView`，只需要 `view.layer.cornerRadius = 10.0;` 就可以设置圆角。只设置 `cornerRadius` 是不会引发离屏渲染的。

对于 `UILabel`, `UIImageView`，还需要设置 `label.layer.masksToBounds = YES;` 才可以，且会引发离屏渲染。

推荐的方式：

`UILabel` 添加圆角

```objc
    // 背景色圆角，注意是 label.layer.backgroundColor
    UILabel *label = [[UILabel alloc]initWithFrame:CGRectMake(100, 100, 100, 100)];
    label.text = @"Hello World!";
    label.layer.backgroundColor = UIColor.greenColor.CGColor;
    label.layer.cornerRadius = 10.0;
    [self.view addSubview:label];

    // 边框圆角
    UILabel *label2 = [[UILabel alloc]initWithFrame:CGRectMake(100, 300, 100, 100)];
    label2.text = @"Hello World!";
    label2.layer.borderColor = UIColor.greenColor.CGColor;
    label2.layer.borderWidth = 1.0;
    label2.layer.cornerRadius = 10.0;
    [self.view addSubview:label2];
```

`UIImageView` 添加圆角

```objc
@implementation UIImageView (CornerRadius)
- (void)addCornersWithRadius:(CGFloat)radius {
    UIGraphicsBeginImageContextWithOptions(self.bounds.size, false, UIScreen.mainScreen.scale);
    CGContextRef context = UIGraphicsGetCurrentContext();
    UIBezierPath * path = [UIBezierPath bezierPathWithRoundedRect:self.bounds byRoundingCorners:UIRectCornerAllCorners cornerRadii:CGSizeMake(radius, radius)];
    CGContextAddPath(context, path.CGPath);
    CGContextClip(context);
    [self.image drawInRect:self.bounds];
    UIImage *output = UIGraphicsGetImageFromCurrentImageContext();
    UIGraphicsEndImageContext();
    self.image = output;
}
```

# hitTest

视图结构：MyViewController - ParentView - ChildView

点击 ChildView：

```
ParentView hitTest
ParentView point(inside:)
ChildView hitTest
ChildView point(inside:)
ParentView hitTest
ParentView point(inside:)
ChildView hitTest
ChildView point(inside:)
LifeViewController touchesBegan
ParentView touchesBegan
ChildView touchesBegan
```

如果 childView.isUserInteractionEnabled = false

```
ParentView hitTest
ParentView point(inside:)
ChildView hitTest
ParentView hitTest
ParentView point(inside:)
ChildView hitTest
LifeViewController touchesBegan
ParentView touchesBegan
```

`hitTest(_:with:)` This method traverses the view hierarchy by calling the `point(inside:with:)` method of each subview to determine which subview should receive a touch event. If `point(inside:with:)` returns true, then the subview’s hierarchy is similarly traversed until the frontmost view containing the specified point is found.
