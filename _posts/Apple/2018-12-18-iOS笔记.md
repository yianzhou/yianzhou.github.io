xcconfig 的配置
1. 新建 Debug.xcconfig 和 Release.xcconfig 文件。
2. Open your project in Xcode, and select the project file in the Xcode Navigator. Select project from the Select a project or target dropdown.
3. 配置 Configurations 为我们新建的 xcconfig。
