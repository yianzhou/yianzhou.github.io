官网：<https://plantuml.com/>

下载 Chrome 插件：PlantUML Viewer Offered by: Peter Prikryl.

在 Chrome 插件选项中，选中“Allow access to file URLs”。

编辑文本保存为`test.txt`：

```
@startuml
Bob->Alice:Hello
@enduml
```

在 Chrome 中打开这个文件，完成！

VSCode 下载 PlantUML 插件，代码可以格式化。
