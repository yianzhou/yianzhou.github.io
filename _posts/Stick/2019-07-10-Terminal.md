# 命令
查看当前路径：`pwd`

在当前目录下根据名称查找文件：`find ./ -name "default_white.bundle"`

查看文件历史：`git log .//Simeji/SimejiCoreIntl/Resources/keyboardskin/default_white.bundle`

# 开启自动补全

Type in terminal `vim ~/.inputrc`
Paste the following on separate lines:
`set completion-ignore-case on set show-all-if-ambiguous on TAB: menu-complete`
