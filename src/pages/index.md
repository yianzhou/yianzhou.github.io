仓库要有两个分支，`master` 存放源代码，`gh-pages` 存放构建输出，用于发布。

要到 Github 的[设置页面](https://github.com/yianzhou/yianzhou.github.io/settings/pages)，将构建分支选择为 `gh-pages` 才可以。
