格式化插件：Prettier - Code Formatter

Shellscript 格式化：Bash Beautify

保存时自动格式化：Preferences -> User Settings

```
{
    "editor.formatOnType": false,
    "editor.formatOnSave": false
}
```

重置所有设置：

1. 打开设置
2. 右上角 - 打开 settings.json
3. 删掉所有内容即可

打开 Terminal：`Control + ~`
