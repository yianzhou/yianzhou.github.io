CLI（Command-Line Interface，命令行界面）：纯文本交互，用户输入命令、程序输出结果，没有任何界面元素，例如 `git`、`curl`。

TUI（Text-based User Interface，文本用户界面）：在终端里用字符绘制出窗口、面板、按钮、滚动条等可视化界面，支持键盘快捷键导航，例如 `vim`、`htop`。简单来说，TUI 是「终端里的 GUI」。

## Token

绝大多数用户对token的消耗、模型能力和定价机制都缺乏足够的了解，也被“Agent时代来临”的宏大叙事裹挟，急于为自己不需要的能力埋单。

OpenAI将GPT-5.4输入价格涨至2.5美元/百万token，输出价格15美元/百万token。

OpenRouter是一个聚合多家AI大模型的API中转平台

1个汉字约等于0.6个token，一个英语单词约当于1.3个token

而涨价的原因，依然是“老生常谈”——供需不平衡。token需求是指数级的，但供给侧的增长是线性的。因为芯片制造、数据中心建设、电力供应，都是重资产、长周期的基础设施投资，即使资本投放翻倍，产能的释放也需要时间。
