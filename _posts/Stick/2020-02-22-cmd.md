# 开启自动补全

Type in terminal `vim ~/.inputrc`

Paste the following on separate lines: `set completion-ignore-case on set show-all-if-ambiguous on TAB: menu-complete`

# Git

tag 某个 commit：`git tag r20191211_3.4.4 ff5aeaa`

To push a single tag: `git push origin r20191211_3.4.4`

push branch to remote: `git push -u origin branchname`

Delete tag on remote: `git push --delete origin tagname`

# 命令

杀死进程：打开`Activity Monitor`，查看PID，在终端中：`kill -9 [pid]`

查看当前路径：`pwd`

在当前目录下根据名称查找文件：`find ./ -name 'default_white.bundle'`

在用户文件夹下查找目录：`find ~ -name 'Autosaved Information' -type d`

查看文件历史：`git log .//Simeji/SimejiCoreIntl/Resources/keyboardskin/default_white.bundle`

# 清理磁盘空间

`rm -rf ~/Library/Developer/Xcode/iOS\ DeviceSupport`

# Proxy

测试：`curl -i https://google.com`

Set proxy: `export all_proxy=http://10.249.38.21:8128`

Set socks5: `export all_proxy=socks5://127.0.0.1:1086`
